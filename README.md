### Технологии
- FastAPI - веб-фреймворк
- MongoDB - база данных
- Pillow - генерация изображений
- Pydantic - валидация данных

### Список эндпоинтов API:

##### 1. `POST /rabbits/`

**Создать нового кролика**
- Принимает: `nickname` (обязательно), `genom` (опционально), `parents` (опционально)
- Если геном не указан, генерирует случайный (4 цифры 0-9)
- Возвращает: данные созданного кролика в JSON
##### 2. `GET /rabbits/{rabbit_id}`

**Получить информацию о кролике**
- Возвращает: данные кролика по ID или 404 если не найден
##### 3. `POST /rabbits/breed/`
**Скрестить двух кроликов**
- Принимает: `mother_id` и `father_id`
- Проверяет, что мать - самка (код 400 если нет)
- Возвращает: список потомков (1-3 детеныша) в JSON
##### 4. `GET /rabbits/{rabbit_id}/family-tree`

**Получить семейное древо (JSON)**
- Параметры: `depth` (глубина древа, по умолчанию 3)
- Возвращает: структуру древа в JSON
##### 5. `GET /rabbits/{rabbit_id}/family-tree-png`
**Получить семейное древо (изображение)**
- Генерирует ASCII-древо → конвертирует в PNG → возвращает base64
- Использует: `EnhancedFamilyTreeArtist` + `AsciiToImageConverter`
##### 6. `GET /rabbits/{rabbit_id}/image/`
**Получить изображение кролика**
- Генерирует ASCII-представление → конвертирует в PNG → возвращает base64
- Использует: `AsciiToImageConverter`
